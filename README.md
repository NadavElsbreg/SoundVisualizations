# SoundVisualizations
sound vis is a tool to take or record audio files and get a video in a 9x16 resolotion video that visualize the sound waves.
ready for upload for social media.

how to get ready:
simply open the "Visualization.html" file in your browser.
curntly only tested for:
 - chrome 141.0.7390.123
 - opera gx

How to use:
 - to start recording first press the "Start Mic" button, this would ask you premision to use your mic(approve the meesege).
 - to upload a file to visualize, press "Choose File", this would ask you to pick a file. when chosen, press the "Load File", this would start running the file and visualize it.
 - when chosen the input sound, start recording with "Start Recording" and finish it with the stop botton.

When finished the output would show lower on the page.
At this point you export it and choose a name for the file.

If wished, the colors and width of the lines can be adjusted up above.
